# Power BI Dashboard

This folder contains the Power BI visualization layer of the project.

The dashboard is built on top of the Gold Layer from the data warehouse and provides interactive insights into league and team performance.

## Features

- KPI summary metrics
- League comparison
- Team rankings
- Drill-through navigation
- Interactive filtering
